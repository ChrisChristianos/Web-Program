Git Guide

Εγκατάσταση του Git:

Πατήστε εδώ για να κατεβάσετε το Git.

Δημιουργία λογαριασμού στο GitHub .


Εντολές στο Git:

git clone URL   :   Συνδέει το local directory με το λογαριασμό του GitHub.
ls   :   εμφανίζει τα αρχεία που έχουμε στο GitHub.
git status   :   μας δείχνει τις ενέργειες που δεν έχουμε αποθηκεύσει.
git add index.html   :   προσθέτει το αρχείο index.html.
git commit -m"comments here"   :   ετοιμάζει να προωθήσει αρχεία στο GitHub και αφήνει μήνυμα για τις αλλαγές.
git push   :   προωθεί τα αρχεία στο GitHub.
git pull   :   αποθηκεύει στο local directory τα αρχεία που υπάρχουν στο GitHub μαζί με τις τροποποιήσεις.
